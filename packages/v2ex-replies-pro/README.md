# 简介

因为 V2EX 的主题回复之间没有关联，所以想知道发生了什么还得 Ctrl + F 搜索被艾特的人说了什么。

点击“@xxx”的链接即可打开右侧“xxx 说了什么？”的弹窗，而且弹窗中的“@xxx”仍可点击。
